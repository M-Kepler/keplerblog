- 安装 pip3

  ```s
  sudo apt-get install python3-pip
  ```

- 安装 git

  ```s
  sudo apt-get install git
  ```

- 安装 virtualenv

  ```s
  sudo pip3 install virtualenv
  ```

- 安装 `nginx` 和 `uwsgi`

  参考博客里的'里程碑#1',我在安装了 nginx 和 uwsgi 后没有出现,

  是因为没有放行 80 端口,去到安全组那里新建一条规则放行 80 端口后访问主机外网地址，ok
